# AI Code Review Agent

A highly interactive, premium React application designed to visualize AI-driven code reviews for GitHub Pull Requests. It integrates seamlessly with an n8n webhook workflow to provide line-by-line feedback right alongside your source code.

## 🚀 Features

- **Dynamic PR Analysis**: Simply paste a GitHub PR link. The UI triggers a backend n8n workflow to generate a comprehensive AI code review.
- **GitHub-Style Inline Comments**: Dynamically downloads the raw source code of the modified files from GitHub and injects the AI's comments specific to the exact line number, simulating a real GitHub review experience.
- **Severity Color-Coding**: Visually categorizes AI insights into `CRITICAL` (Red), `MAJOR` (Orange), `MINOR` (Blue), and `SUGGESTION` (Purple) using visually distinct badges and line-highlighting.
- **Rich Markdown Formatting**: Safely parses and renders Markdown generated by LLMs (including code blocks, tables, and lists) using `react-markdown` and `remark-gfm`.
- **Premium UI**: Built with modern Tailwind CSS featuring glassmorphism, animated states, scroll synchronization, and a beautiful light-purple overarching aesthetic.

## 🛠️ Architecture & Tech Stack

This project is entirely decoupled from traditional GitHub webhooks. 

**Frontend:**
- **React** (Functional Components + Hooks)
- **Vite** (Lightning-fast Build Tool)
- **Tailwind CSS** (Utility-first styling, Animations, Typography)
- **react-markdown** & **remark-gfm** for parsing LLM Markdown.

**Backend Simulation:**
- **n8n Webhook**: The frontend triggers an HTTP POST request to a running n8n instance (using ngrok for local tunneling). 
- n8n takes the `pr_link`, talks to GitHub via Personal Access Token, prompts an LLM (Claude/OpenAI), and returns a structured JSON (`summary` and `comments` array) back to the UI.

## ⚙️ Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- A running n8n instance with a webhook configured at your target URL.

### Installation

1. Clone the repository and navigate into it.
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
4. Open your browser to the local port provided by Vite (usually `http://localhost:5173`).

### Configuration
By default, the application points to a preconfigured ngrok webhook URL. To connect this to your own n8n instance:
1. Open `src/App.jsx`.
2. Locate the `triggerReview` function.
3. Replace the `fetch` URL with your own webhook address.

## 🧠 The Agent Approach
This repository was developed utilizing autonomous AI coding agents adhering strictly to guidelines in the `.agents` directory:
- `SKILLS.md`: Defined the exact UI topology (Header, SummaryPanel, CodeView).
- `mistakes.md`: Persisted memory of previous failures (e.g., Markdown handling, state mixing) to prevent regression.
- `workflows/`: Provided step-by-step instructions for adding specific features like the PR link input state.
